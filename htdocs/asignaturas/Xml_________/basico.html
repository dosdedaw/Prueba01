  2. Etiqueta META

    3. Dentro de la cabecera de una página web (entre <head> y </head>) es donde se coloca las etiquetas de metainformación<meta>. Estas etiqueta sirven para, entre otras cosas, indicar a los motores de búsqueda que hay en Internet la información que queremos que se almacene en estos; por ejemplo el título de la página, temática de esta o las palabras por las cuales deseamos que se localice nuestra página en Internet. Los atributos que se pueden poner a la etiqueta meta son:

    1. content

      2. Este atributo siempre va emparejado con el atributo name o con http-equiv y contendrá un texto que se corresponderá con el valor del atributo al que acompaña.

    2. name

      3. Este atributo sirve para indicar el tipo del elemento meta que se va a utilizar. Algunos de ellos son:

      1. author

        2. Cuando utilizamos este valor, en el atributo content indicaremos quien es el autor de la página. Normalmente sólo se coloca en la primera página de nuestra web. <meta name=”author” content=”Socorro Manzano”>

      2. generator

        3. Con este valor en el atributo content indicar el nombre y la versión del programa. <meta name=”generator” content=Macromedia Dreamwears 2.01”>

      3. lang

        4. Indicar el idioma del documento. &lymeta name=”lang” content=”es”>&ly!—Idioma del documento (Español)-->

      4. descrption

        5. No debe sobrepasar los 250 caracteres. <meta name=”description” content=”Mi Página”>

      5. keywords

        6. Indicamos las palabras claves por las que queremos que los buscadores encuentren o indexen nuestra página. Las palabras deben ir separadas por comas y a ser posible, sin espacios. <meta name=”keywords” content= ”presentación, recursos, objetivos,sitio web”>

    3. http-equiv

      4. Este atributo puede utilizarse en lugar de name y posee un significado especial cuando los documentos se obtienen por medio de http. Los servidores utilizan el nombre de la propiedad especificada en el atributo para http-equiv para crear una cabecera en la repuesta HTTP

      1. Content-Type

        2. ndica el tipo de documento elaborado y el juego de caracteres utilizado. El tipo de caracteres que utiliza el documento es el denominado Latin1, cuyo código ISO es el 8859-1 <meta http-equiv=”Conent-Type” content=”text/html;charset=iso-8859-1”>

      2. Refresh

        3. Cargue una determinada página después de que pase un tiempo especificado por nosotros.
        <meta http-equiv=”Refresh” content=”10;URL=http://www.google.es”>

      1. Elemento link

        2. El elemento link no tiene contenido pero conlleva información sobre la relación entre el documento actual y el documento o recurso utilizado. Uno de sus atributos es rel, que nos permite definir la relación entre diversos documentos
        <link rel=”Next” ref.=”documento.html”>

      2. El formato

        1. El formato de texto lo componen las siguientes etiquetas y atributos:
        2. Etiqueta
          1. <font>

            2. Tiene tres atributos y necesita llevar al menos uno para que funcione. Tambien necesitara ser cerrado </font>

            1. size

              2. Número del 1 al 7. Tamaño de la letra entre 8 y 20 puntos.

            2. face

              3. Nombre de la fuente. Tipo de fuente, arial, courier…, Si ponemos más de una fuente separadas por comas, en este atributo, se cargará la primera de ellas siempre y cuando ésta se halle en el sistema operativo del usuario que ve la página. Si la fuente indicada( o ninguna de las fuentes especificadas) no existe en SO del internauta, se verá la fuente que tenga por defecto su navegador, normalmente Times New Roman

            3. color

              4. Este atributo especifica el color del texto. Nombre de un color o su equivalente en código hexadecimal. Hay 140 nombre de colores, o en hexadecimal (#RRGGBB). Las dos primeras letras indican la cantidad de rojo, las dos siguientes verde y las dos últimas azul.

        3. <basefont>

          4. Establece el tamaño base de fuente (usando el atributo size). Es un elemento que no tiene etiqueta de fin. Los cambios del tamaño de la fuente producidos con <font> son relativos al tamaño base de fuente establecido por <basefont>. Si no se usa <basefont>, el tamaño base de fuente es el indicado en el atributo.

      3. Formato de párrafo

        1. <p></p>

          2. Crea un nuevo párrafo haciendo un salto de línea, es decir introduciendo una línea en blanco entre el párrafo que precede a dicho código y el texto situado a continuación. Dentro de un párrafo hay que obligar al salto de línea.
          <p align=”center”> texto </p>

          1. align

            2. Atributo opcional, que puede tomar los valores

            1. right

              2. justificar el texto a la derecha

            2. left

              3. justificar el texto a la izquierda

            3. center

              4. texto centrado

            4. justify

              5. justificar el texto a ambos lados

        2. <div></div>

          3. Sirve para alinear elementos sin necesidad de crear un nuevo párrafo tiene un atributo obligatorio align. No hace salto de línea. <div align=”center”> Texto </div>

          1. align
            1. right

              2. justificar el texto a la derecha

            2. left

              3. justificar el texto a la izquierda

            3. center

              4. texto centrado

        3. <center></center>

          4. Sirve para centrar elementos en las páginas, sin tener que usar <div align=”center”> o <p align=”center”>

        4. <blockquote></blockquote>

          5. Hace que el párrafo se sangre por ambos lados, izquierda y derecha.

        5. <br><br/>

          6. Salto de línea, sin necesidad de crear un nuevo párrafo.

        6. <nobr>

          7. Evita que el navegador haga un salto de línea en el lugar donde NO nos gustaría, como puede ser Banco de Santander. Banco de Santander <nobr>

        7. <pre></pre>

          8. Nos guarda la estructura del texto escrito en el editor, respetando los espacios en blanco tabuladores y retorno de carro. El uso del preformato nos garantiza que una línea de texto original no se verá partida en dos o más líneas en el cliente web.

        8. <h1></h1>....<h6></h6>

          9. Sirve para poner un tamaño grande (más o menos 20 puntos) y estilo negrita a todo el texto del párrafo, como si de un título se tratase. Admite el atributoaligncon los mismo valores que la etiqueta<p>
          Al contrario que los size aqui a numero menor mayor la letra (<h1> es la letra mas grande).

      4. El formato de la página

        5. El formato de la página es aquel que se aplica a todo el documento. En este caso no lo conforman las etiquetas, sino los atributos de la etiqueta <body>. Mediante estos atributos podemos cambiar el color por defecto del navegador: color del texto y los diferentes hiperenlaces. Los atributos son:

        1. text

          2. El atributo text debe ser igual a un color, ya sea en código hexadecimal o con nombre del color. El color elegido debe contrastar con el color que utilice la fuente. El color aquí elegido será el de la fuente por defecto, si queremos cambiarlo hay que utilizar la etiqueta <font color=”colorelegido”> <bodytext=”red”> el color de la fuente será el rojo.

        2. link (vinculo)

          3. Este atributo también será igual a un color, pero sólo sirve para poner color al texto o borde de la imagen cuando estos sean hiperenlaces. Si no se pone nada suele salir en color azul, que son los colores por defecto de nuestro navegador.

        3. vlink(vinculo visitado)

          4. Sirve para controlar el color de los vínculos visitados

        4. align(vinculo activo)

          5. Sirve para controlar el color del vinculo activo.

        5. bgcolor(color de fondo)

          6. Pone un color de fondo a todo el documento.

        6. Background(fondo)

          7. Sirve para poner una imagen de fondo de la página. No es necesario que la imagen ocupe toda la superficie, pues el navegador creará un efecto de mosaico para repartir la imagen por toda la página.

          Si ponemos los atributos bgcolor y blackground en la etiqueta , siempre prevalecerá el blackground sobre bgcolor, aunque este segundo se cargara primero. La imagen que se ponga de fondo puede tener los formatos jpg,gif, png. <body bgcolor=”silver” background=”imagren.jpg”>
